# yt-in-mp3 [![Build Status](https://travis-ci.org/welksonramos/yt-in-mp3.svg?branch=master)](https://travis-ci.org/welksonramos/yt-in-mp3) [![npm version](https://img.shields.io/npm/v/yt-in-mp3.svg)](https://www.npmjs.com/package/yt-in-mp3)
>Convert Youtube videos to mp3 using [youtubeinmp3](http://youtubeinmp3.com)

## Installation

```bash
$ npm install -g yt-in-mp3
```

## How to use

```bash
$ yt-in-mp3 --help

  Usage
   $ yt-in-mp3 <url>
 
   Example
   $ yt-in-mp3 https://www.youtube.com/watch?v=QH2-TGUlwu4
```
